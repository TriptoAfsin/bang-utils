## Description

bang-utils is a lightweight utility library for converting English numerals and dates to Bengali in TypeScript/ JavaScript. It provides functions to convert numbers, dates, and months from English to Bengali script.

### Features -

1. Full TypeScript Support
1. English Number to Bangla Number(21.21 -> рзирзз.рзирзз)
1. English Date to Bangla Date(2-5-2024 -> рзл ржлрзЗржмрзНрж░рзБрзЯрж╛рж░рж┐, рзирзжрзирзк)
1. English Time to Bangla Time (15:48 -> ржжрзБржкрзБрж░ рзй ржЯрж╛ рзкрзо ржорж┐ржирж┐ржЯ)
1. English Number to Bangla In Words(50.25 -> ржкржЮрзНржЪрж╛рж╢ ржжрж╢ржорж┐ржХ ржжрзБржЗ ржкрж╛ржБржЪ)
1. Currency Number to Bangla Currency(125.58 -> рззрзирзл ржЯрж╛ржХрж╛ рзлрзо ржкрзЯрж╕рж╛)

## Installation

You can install bang-utils via npm or yarn:

```bash
npm install bang-utils
```

or

```bash
yarn add bang-utils
```

## Usage

### Importing

You can import the functions from bang-utils using ES6 import syntax:

```javascript
import {
  numBang,
  dateBang,
  monthBang,
  numToBanglaWords,
  dayBang,
  currencyToBang,
  timeBang,
} from "bang-utils";
```

### Converting English Numerals to Bengali

```javascript
console.log("number: ", numBang(21)); // Output: number: рзирзз
console.log("decimal number: ", numBang(21.21)); // Output: decimal number: рзирзз.рзирзз
```

### Converting Dates to Bengali

```javascript
console.log("month: ", monthBang("2024-05-02")); // Output: month: ржорзЗ
console.log("day: ", dayBang("2024-02-26")); //day: рж╕рзЗрж╛ржоржмрж╛рж░
console.log("date: ", dateBang("2-5-2024", ",", "D M, YYYY")); // Output: date: рзл ржлрзЗржмрзНрж░рзБрзЯрж╛рж░рж┐, рзирзжрзирзк
```

### Converting Numbers to Bengali Words

```javascript
console.log("to Bangla words: ", numToBanglaWords(50.25)); // Output: to Bangla words: ржкржЮрзНржЪрж╛рж╢ ржжрж╢ржорж┐ржХ ржжрзБржЗ ржкрж╛ржБржЪ
```

### Converting Currency Numbers to Bengali Currency

```javascript
console.log("currency: ", currencyToBang(50)); //currency: рзлрзж ржЯрж╛ржХрж╛
console.log("currency: ", currencyToBang(125.58)); //currency: рззрзирзл ржЯрж╛ржХрж╛ рзлрзо ржкрзЯрж╕рж╛
```

### Converting Date Time to Bengali Time

```javascript
console.log("detailed time: ", timeBang(new Date())); //ржжрзБржкрзБрж░ рзй ржЯрж╛ рзкрзо ржорж┐ржирж┐ржЯ
console.log("normal time: ", timeBang(new Date(), "normal")); // ржжрзБржкрзБрж░ рзй.рзлрзж ржорж┐ржирж┐ржЯ
console.log("short time: ", timeBang(new Date(), "short")); //рзй.рзлрзж ржорж┐ржирж┐ржЯ
```
